How the templates work

  • Every section defaults to a normal/preserved sentence — only abnormal findings need clicks.
  • Grade buttons (Normal / G1 / G2 / partial / etc.) are mutually exclusive — pick one per section.
  • Location buttons that say (pick one or more) are multi-select. Grammar adjusts automatically: medial + lateral facets → "the medial and lateral facets" (auto-pluralized).
  • Green-check addons (subchondral marrow edema, paralabral cyst, etc.) are toggle additions and append to the section's sentence.
  • Numeric inputs (size in mm, retraction) flow into {placeholders} in the sentence template.
  • Bracketed text like [anatomic location] = unfilled placeholder — pick a location to replace it.
  • Copy all = entire compiled report. Copy abnormal only = skips normal sentences (when PowerScribe already has default normal text).
  • Sections are grouped in output by anatomic region per the MSK protocol PDF search-pattern order.

Templates are a starting point — edit the inline window.TEMPLATE_CONFIG in each joint's HTML file to refine sentences, add options, or extend coverage.
